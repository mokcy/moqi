#Moqi Mobiel CMS (Version: freerice)

##Front-end DEPENDENCIES
 - Backbone
 - RequireJS
 - Underscore
 - Mustache
 - Pure CSS

##Back-end DEPENDENCIES

 - RESTify

##Test DEPENDENCIES

 - Jasmine
 - Chai
 - Sinon
 - Mocha
 - Jasmine-jquery

##Quick Start

1. Run ``npm install``
2. Run ``db-migrate up``
3. Run ``node server/app.js`` (only:production)
